# Desafio Front-end Delfosim

- **Node** (12.13.0 ou alguma versão superior)
- **NPM ou Yarn**

### Instalando Dependencias do Porojeto

Na raiz do projeto basta abrir o terminal e digitar `npm install` ou `yarn` que logo em seguida ira instalar todas as dependencias necessarias para rodar o projeto

### Iniciar Servidor

Logo após as dependencias necessarias do projeto, abra o terminal na raiz do projeto e execute **`npm run start` ou `yarn start`** para iniciar o servidor no endereço [localhost:3000](http://localhost:3000)
